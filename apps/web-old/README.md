# Employee Knowledge Hub

An internal dashboard for employees to access company-wide information, including documents, meeting notes, and project details.

## Features

-   **Dashboard:** A central hub for quick access to important information.
-   **Documents:** A searchable repository of company documents.
-   **Meetings:** A log of past and upcoming meetings.
-   **Projects:** An overview of ongoing projects.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
